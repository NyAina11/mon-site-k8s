Fonctionnalités

Ce que le projet fait

Un système complet de déploiement piloté depuis un téléphone Android ou iOS.

📊
Dashboard temps réel
Visualise l'état de tous les Deployments, pods et certificats SSL du cluster en un coup d'œil.
🚀
Déploiement en 1 tap
Formulaire mobile → Deployment + Service + Ingress + Certificate créés automatiquement dans Kubernetes.
◎
Deploy depuis GitHub
Entre l'URL d'un repo GitHub avec un Dockerfile → build, import dans k3s et déploiement automatiques.
🔒
SSL automatique
Cert-Manager provisionne et renouvelle les certificats TLS automatiquement. HTTPS actif sans aucune config manuelle.
📋
Logs en streaming
Stream WebSocket des logs de n'importe quel pod en temps réel, avec coloration syntaxique par niveau.
🛡️
Auth JWT + RBAC
Authentification par token JWT. Le backend tourne avec un ServiceAccount K8s aux droits strictement nécessaires.
